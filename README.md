Project ini merupakan sebuah kalkulator modern berbasis HTML, CSS (Tailwind), dan JavaScript yang dirancang dengan tampilan bergaya glassmorphism berwarna pink agar terlihat elegan dan responsif di berbagai ukuran layar. Kalkulator ini mendukung operasi aritmatika dasar seperti penjumlahan, pengurangan, perkalian, dan pembagian, serta dilengkapi fitur tambahan seperti history perhitungan yang menyimpan hingga 20 riwayat dan dapat diklik kembali untuk digunakan, serta tombol CLEAR untuk menghapus seluruh history. Selain itu, kalkulator ini juga memiliki fungsi memori seperti MC, MR, M+, dan Mâˆ’ yang bekerja layaknya kalkulator profesional, lengkap dengan indikator memori yang muncul ketika nilai tersimpan. Sistemnya juga mendukung kontrol penuh melalui keyboard, termasuk input angka, operator, Enter untuk hasil, Backspace untuk hapus satu digit, dan Esc untuk menghapus semua.
